Supplementary Data Files README

This README accompanies the structured supplementary data files for "An Explainable Machine Learning Framework for Hypothesis Generation in Biochemical Methane Potential Prediction."

Journal-uploaded files:
- Supplementary_Dataset_Row_Level_OOF_Predictions.csv: row-level dataset for the 127 eligible feedstocks, including source descriptors, model features, observed BMP per VS, reconstructed BMP per DM, target-construction fields, mean repeated out-of-fold predictions, prediction errors, residual columns, eligibility fields, and labeled apparent full-data fitted predictions.
- Supplementary_Dataset_Data_Dictionary.csv: data dictionary defining each column, unit, prediction type, sign convention, and provenance category.
- Supplementary_Dataset_README.pdf: this README in PDF form, supplied separately to the journal. The repository contains Supplementary_Dataset_README.txt as a plain-text counterpart; the TXT counterpart is not a fourth journal-uploaded supplementary data file and no README PDF is expected inside the repository supplementary_data_files directory.

Source data citation:
Lallement A, Peyrelasse C, Lagnet C, Barakat A, Schraauwers B, Maunas S, Monlau F. (2023). A Detailed Database of the Chemical Properties and Methane Potential of Biomasses Covering a Large Range of Common Agricultural Biogas Plant Feedstocks. Waste, 1(1), 195-227. https://doi.org/10.3390/waste1010014

Model-scenario key:
- A_BMP_VS_all_features: BMP per VS sensitivity scenario using all model predictors.
- B_BMP_DM_no_DM_VS: BMP per DM sensitivity scenario excluding DM and VS predictors.
- C_BMP_DM_all_features: primary BMP per DM Random Forest model using all model predictors.
- D1_BMP_DM_no_DM: BMP per DM sensitivity scenario excluding DM.
- D2_BMP_DM_no_VS: BMP per DM sensitivity scenario excluding VS.

Prediction conventions:
- Out-of-fold (OOF) predictions are validation predictions generated only when a row was excluded from model training.
- Apparent full-data fitted values are generated by models fitted on all eligible rows and are not validation predictions.
- Prediction error is predicted minus observed; positive values indicate overprediction.
- Residual columns are observed minus predicted.
- Metrics recalculated globally from row-level mean OOF predictions may differ slightly from mean outer-fold metrics because the manuscript reports repeated nested-CV outer-fold summaries.

Primary performance reference:
The primary nested Random Forest estimate for C_BMP_DM_all_features was R2 = 0.530, RMSE = 44.5 Nm3 CH4/t DM, and MAE = 33.6.

Interpretation note:
The Lignin-DM pattern is exploratory, modest, model-derived, and non-causal; it should be used for hypothesis generation rather than mechanism or deployment claims.

Repository-only files:
Fold assignments, individual repeated OOF predictions, selected hyperparameters, GridSearchCV candidate results, held-out SHAP outputs, scripts, environment files, and generated detailed outputs are provided only in the public reproducibility repository.
